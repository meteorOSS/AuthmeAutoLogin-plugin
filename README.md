# AuthmeAutoLogin-plugin

### 为使用AuthMe的服务器提供了更简洁的登陆流程;
### 去除主菜单，直接输入密码进入指定的服务器 (新用户将自动注册)

模组基于 1.18 开发
安装前请确保使用的是 [AuthMeReloaded](https://github.com/AuthMe/AuthMeReloaded)
